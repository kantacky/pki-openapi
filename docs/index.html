PKI Certificate Management API (1.0.0)

Download OpenAPI specification:

PKI Certificate Management Team: pki@kantacky.com License: MIT

X.509証明書管理システムのためのRESTful API。 証明書の発行、管理、検証、およびCA（認証局）の管理機能を提供します。

証明書一覧取得

証明書の一覧を取得します。フィルタリング、ソート、ページネーションをサポートします。

Authorizations:
ApiKeyAuthOAuth2
query Parameters
page
integer >= 1
Default: 1

ページ番号

size
integer [ 1 .. 100 ]
Default: 20

ページサイズ

status
string
Enum: "active" "revoked" "expired" "pending"

証明書の状態でフィルタ

subject
string

主体者名で部分一致検索

issuer
string

発行者名で部分一致検索

validFrom
string <date>

有効開始日以降の証明書を検索

validTo
string <date>

有効終了日以前の証明書を検索

sort
string
Default: "createdAt"
Enum: "validFrom" "validTo" "createdAt" "subject"

ソート順

order
string
Default: "desc"
Enum: "asc" "desc"

ソート方向

証明書発行

ApiKeyAuthOAuth2
Request Body schema: application/json
required
subject
required
string

証明書の主体者（Distinguished Name）

subjectAltNames
Array of strings

サブジェクト代替名

keySize
required
integer
Default: 2048
Enum: 2048 3072 4096

鍵長（ビット）

keyType
string
Default: "RSA"
Enum: "RSA" "EC"

鍵アルゴリズム

curve
string
Enum: "P-256" "P-384" "P-521"

楕円曲線（ECの場合）

validityDays
integer [ 1 .. 3650 ]
Default: 365

有効期間（日数）

keyUsage
Array of strings

extendedKeyUsage
Array of strings

string <uuid>
